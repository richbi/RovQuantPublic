# fagrapport74_Wolverine2013-2021

Repository containing the input data, R script, figures, tables and raster density maps from the following technical report: 
Milleret, C., Dupont, P., Brøseth, H., Flagstad, Ø, Kindberg, J., and Bischof, R., 2022. Estimates of wolverine density, abundance, and population dynamics in Scandinavia, 2013 - 2021 - MINA fagrapport 74. 30pp.
